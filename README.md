
# Cineverse

Flutter Application for search movies from API TMDB.

## Fitur

- On Boarding
- Home Screen
- Search Movie by Keyword
- Search Movie by Genres
- Detail Movie (Trailer, Synopsis, Reviews, Cast)
- ...(coming soon)
